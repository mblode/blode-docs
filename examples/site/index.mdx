# Knowledge OS

A Git-first workspace for docs, blogs, slides, forms, and business content.

## What it is

- One repo
- One config
- One workflow

## Why it works

- Plain text you can version and review
- MDX for rich content
- Fast local preview

## Get started

Run:

```bash
npx knowledge init
```

Then edit this file and refresh your preview.
